このページは共用フロントです。写真や観察データは、設定した自分のPCにだけ送信されます。

Start

使い始める順番

1. 設定 必須

設定 で自分のPCの Tailscale HTTPS URL とアプリパスワードを保存します。

2. 追加 写真準備

追加 で写真を選び、PC停止中でも「あとで送信する」で端末保存できます。

3. 未送信 手動送信

未送信 から、自分のPCが起動しているときだけ送信します。
