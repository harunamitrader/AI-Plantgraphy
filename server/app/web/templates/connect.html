Steps

外出先で使う準備

  1. PCとスマホにTailscaleを入れます。
  2. 両方で同じTailscaleアカウントにログインします。
  3. PCでAI Plantgraphyを起動したままにします。
  4. スマホのTailscaleをONにして、HTTPSのQRコードを読みます。

詳しい手順はリポジトリ内の `docs/TAILSCALE_SETUP.md` にあります。

Home screen

ホーム画面に追加

スマホのブラウザメニューから「ホーム画面に追加」を選ぶと、AI Plantgraphyをアプリのように開けます。

本格的なPWAインストール表示と連続カメラはHTTPS接続で有効になります。Tailscale ServeのHTTPS URLを使ってください。
